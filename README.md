# Cursor 免费助手

## 简介

完全免费的 Cursor 助手工具，提供以下功能：
- 一键获取新额度
- 自动满额度账号
- 无需登录账号
- 解决机器码问题

**请勿滥用，合理使用**

## 支持平台

支持 Windows、Mac 和 Linux 系统
